Comandos y para que sirven de git
siempre debemos de estar dentro de la carpeta del proyecto
- git init - inicializa git para empezar a trabajar en la carpeta del proyecto
se necesita registrar en consola
- git config --global user.email "correo"
- git config --global user.name "nombre"
se puede ver los archivos que estan en proyecto y que pueden ser agregado en respaldo
- git status
para agregar a ser respaldados
puede ser por archivo
- git add "nombre del archivo sin comillas y su extension"
- git add . "se agregan todos los archivos de forma general"
para poder crear el respaldo
- git commit -m "mensaje descriptivo del respaldo"

para ignorar archivos y carpetas en git se puede agregar un archivo llamado
.gitignore
y dentro del archivo agregar los nombres de los archivos y carpetas que se ignoraran

para ver versiones alternativas
- git branch
para crear versiones alternativas
- git branch "nombre del proyecto alternativo"
para moverse entre versiones
- git checkout "nombre del proyecto alternativo"

para crear respaldos remotos como git hub
se necesita crear cuenta en esta plataforma en web github.com
se crea un repositorio
se accede al repositorio con el siguiente comando
- git remote add origin "url donde se agregara de forma remota"
para subir los archivos al repositorio
- git push -u origin master
una vez echo esto se requiere autenticarte
y empezara a subir el codigo al repositorio.
